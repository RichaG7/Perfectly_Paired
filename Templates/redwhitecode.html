Red vs. White Code

Objective

Building a classification model that would predict whether a wine is red or white based on its physicochemical properties: Fixed acidity, volatile acidity, citric acid, residual sugar, chlorides, free sulfur dioxide, total sulfur dioxide, density, pH, sulphates, and alcohol.

Combining the Red and White Dataset

Adding a new column named “type”, and adding “0” or "1" as values to indicate whether the wine is red or white, respectively.
Concatenating the red and white datasets for our analysis.

Exploring the Combined Dataset

Grouping the dataset by “type” to further examine the other columns (physiochemical properties), and seeing if we find any difference between the two groups.

Mainly, we looked into the averages of each column.
Conducted a test to determine if there is a statistical difference for each physicochemical property between red and white wine.

The box plot above shows the data for volatile acidity, but the other physicochemical properties were all significantly significant (p < 0.05) as well.

Building the Deep Learning Model

Created a Deep Learning model – 11 inputs, with 2 layers of 100 hidden nodes, and 2 outputs (red: 0, white: 1).

Achieved a model with really high accuracy (close to 1), so there was no need for extra training.

Saved the model as a “h5” file that we loaded into Flask.

Feature Importance

Using Random Forest, we calculated feature importance to determine which features are the most important in predicting a wine's color - red vs. white.
